## Stack Tecnológico

SGIVU está construido con tecnologías modernas listas para producción:

- **Backend**: Spring Boot 4.0, Spring Cloud, Spring Security OAuth2
- **Frontend**: Angular 21 con TypeScript
- **Servicio ML**: FastAPI, scikit-learn, XGBoost
- **Bases de Datos**: PostgreSQL, Redis
- **Infraestructura**: Docker, Docker Compose, Nginx
- **Observabilidad**: Actuator, Micrometer, Zipkin

## Aspectos Destacados de la Arquitectura

<Steps>
  <Step title="API Gateway con Patrón BFF">
    El gateway actúa como Backend-For-Frontend, gestionando tokens OAuth2 de forma segura con almacenamiento de sesiones en Redis
  </Step>
  <Step title="Descubrimiento de Servicios">
    El registro de servicios Eureka permite la comunicación dinámica entre servicios
  </Step>
  <Step title="Configuración Centralizada">
    Spring Cloud Config Server gestiona la configuración de todos los microservicios
  </Step>
  <Step title="Integración ML">
    El servicio ML basado en FastAPI proporciona APIs de predicción de demanda con capacidades de reentrenamiento de modelos
  </Step>
</Steps>

## Contenido

### Servicios Principales

- **sgivu-auth**: Servidor de Autorización OAuth 2.1/OIDC
- **sgivu-gateway**: API Gateway con patrón BFF
- **sgivu-user**: Gestión de usuarios y roles
- **sgivu-vehicle**: Catálogo de vehículos y gestión de imágenes
- **sgivu-client**: Gestión de relaciones con clientes
- **sgivu-purchase-sale**: Seguimiento de transacciones
- **sgivu-ml**: Servicio de predicción de demanda

### Infraestructura de Soporte

- **sgivu-config**: Servidor de configuración centralizada
- **sgivu-discovery**: Registro de servicios Eureka
- **sgivu-frontend**: Aplicación web basada en Angular
